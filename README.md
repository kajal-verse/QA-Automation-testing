# SauceDemo Selenium Automation Project

This is my first test automation project i built while learning QA testing.

## What i tested
- Login with valid credentials
- Login with invalid credentials
- Locked out user error
- Logout functionality
- Add product to cart
- Remove product from cart

## Tools Used
- Python
- Selenium WebDriver
- Pytest
- WebDriver Manager
- pytest-html (for test reports)
